# Code for Taiwan

## 時間分配

5 分鐘

## 內容

- 標語：一起學習程式思維（Where We Build Programming Thinking）
- 核心理念：概念優於工具、避坑勝過速成
- 社群形式：每月第三個星期三 20:00-21:00 YouTube 線上小聚，完全免費
- 參與對象：工程師、設計師、PM、剛入門的新手，任何對程式有興趣的人
- 與主題連結：從程式思維出發，才能真正駕馭 AI 與 Monorepo

## 完整逐字稿

Code for Taiwan 在做什麼呢？

我們團隊專注在分享程式的思維與觀念。大家應該都有感覺，在這個 AI 盛行的時代，各種 No-code、Low-code 工具讓每一個人都能快速產出內容。乍看之下產值提升了，但背後也藏了一個盲點：如果你不了解程式的邏輯，就算產出了成果，你還是無法真正避坑。

這種感覺就像在玩吃角子老虎機——拉下桿子碰碰運氣，祈禱程式能跑，可是完全不知道為什麼成功，更別說複製成功了。

所以，Code for Taiwan 想做的事情就是專注在「程式觀念」跟「思維」本身。我們不只講技術細節，更希望大家學會判斷力，知道什麼是好的實作、怎麼分析問題。

更重要的是，這個社群不是只為資深工程師而存在。我們非常歡迎 PM、設計師，甚至剛踏入這個領域的新手工程師。只要你想把 AI 工具用得更好、想知道如何避坑，這裡都很適合你。

我們固定在每個月的第三個星期三晚上八點到九點，透過 YouTube 辦線上小聚。活動完全免費，也對所有人開放。你可以把它想像成一個線上客廳，大家換個角度聊程式、分享踩過的坑，從別人的經驗得到靈感。

過去邀請過 zeabur yunalin 分享 vibe-coding 避坑術，五倍學院龍哥帶我們學 n8n，hackMD yukai AI 工具術打造產品思維等等，

我們希望大家帶著「吃角子老虎」的運氣進來，換成可以長期累積的程式思維；帶著對工具的好奇進來，換成真正懂得如何駕馭工具的能力。

如果你也想把寫程式變得更輕鬆、更不踩雷，歡迎加入我們。  
Code for Taiwan：一起學習程式思維，讓工具為我們所用，而不是反過來。
